# Hello World

Welcome to my blog. This is the first post, built with **mdblog** — a minimal static blog generator.

## What is mdblog?

A simple CLI tool that converts MDX files into static HTML. No frameworks, no runtime dependencies.

```ts
deno task build --input ./content --output ./dist
```

That's it.
